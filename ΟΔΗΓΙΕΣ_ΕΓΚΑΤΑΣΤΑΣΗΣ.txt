Σύστημα Διαχείρισης Ιατρικών Ραντεβού
======================================
Ανάπτυξη: Σ. Ντάφλος

ΑΠΑΙΤΗΣΕΙΣ ΣΥΣΤΗΜΑΤΟΣ
-------------------
- PHP (έκδοση 7.4 ή νεότερη)
- MySQL / MariaDB
- Web server (π.χ. μέσω MAMP, XAMPP, WAMP κ.λπ.)

ΟΔΗΓΙΕΣ ΕΓΚΑΤΑΣΤΑΣΗΣ ΚΑΙ ΕΚΤΕΛΕΣΗΣ
--------------------------------
1. Αντιγράψτε τον φάκελο `medical_appointments` στον ριζικό φάκελο του web server σας (π.χ. στο φάκελο `htdocs` για XAMPP/MAMP ή στο `www` για WAMP).
2. Ανοίξτε το phpMyAdmin (ή οποιοδήποτε άλλο εργαλείο διαχείρισης της MySQL).
3. Επιλέξτε "Εισαγωγή" (Import) και ανεβάστε το αρχείο `sql/medical_db.sql` που βρίσκεται μέσα στον φάκελο. Αυτό θα δημιουργήσει αυτόματα τη βάση δεδομένων `medical_appointments`, τους πίνακες, και θα εισάγει κάποια δοκιμαστικά δεδομένα.
4. ΣΗΜΑΝΤΙΚΟ: Το αρχείο σύνδεσης με τη βάση δεδομένων είναι το `config/db.php`. Είναι ρυθμισμένο για MAMP (port: 8889, user: root, pass: root). Αν χρησιμοποιείτε XAMPP, πιθανότατα θα χρειαστεί να αλλάξετε το DB_PORT σε 3306 και το DB_PASS σε κενό (δηλαδή '').
5. Ανοίξτε τον browser σας και μεταβείτε στη διεύθυνση του project. Ανάλογα με τον server σας, συνήθως είναι:
   http://localhost/medical_appointments/
   ή
   http://localhost:8888/medical_appointments/
